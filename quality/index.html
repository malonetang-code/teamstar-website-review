首页/质量体系
QUALITY SYSTEM / ISO 9001:2015

制造全过程质量控制

材料确认、过程检验、尺寸检测与成品检验，依据确认版图纸及相应产品要求执行。

PROCESS CONTROL

质量控制流程

检验方案依据产品、图纸及工况要求制定，覆盖相应的材料、尺寸与刃口标准。

01 / REVIEW

技术要求与材料确认

确认图纸版本、材料路线及适用的检验要求。

02 / IN-PROCESS

过程检验

在刀坯成形、热处理、机加工及研磨过程中实施相应控制。

03 / FINAL

成品检验

检验已确认的材料、尺寸及刃口要求。

04 / RELEASE

批次放行

依据确认标准执行产品放行，并保留适用记录。

CONTROLLED CHARACTERISTICS

依据图纸与工艺路线确定检验项目

围绕材料确认、热处理状态、安装配合、刀具几何及相应表面要求，选择适用的检验方法。

01 / MATERIAL

材料成分确认

根据材料路线和项目要求，选择光谱分析或手持式 X 射线荧光分析等适用方法。

SPECTROLAB 光谱仪、VCA 手持式 X 射线荧光光谱仪
02 / HARDNESS

硬度与金相检验

依据材料、热处理工艺和受控要求，安排维氏硬度、洛氏硬度或金相检验。

三丰 MVK-H100、ZHR-8150LK、NIKON 金相显微镜
03 / DIMENSION

尺寸与几何检测

针对安装接口、孔位、轮廓及其他受控尺寸，选用三坐标、影像测量或投影测量。

三丰 BEYOND APEX 707、KEYENCE IM-7000、JT-300
04 / SURFACE

表面与膜厚检测

按图纸、工艺或验收要求，对适用项目进行表面粗糙度或膜厚检测。

三丰 SJ-310 平面粗度仪、EX-3000 荧光 X 线膜厚计
